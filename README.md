# Структура проекта DS (Data Science & Machine Learning)

В данном репозитории собраны учебные и практические материалы по анализу данных, работе с базами данных SQL, визуализации и машинному обучению с использованием библиотеки **Marimo** и классических библиотек **scikit-learn**, **pandas** и **numpy**.

Проект реорганизован в модульную структуру для удобства навигации и поддержки кода.

## Ключевые разделы подробнее

### 1. Линейная Регрессия (`Dima/ml/linear_regression/`)
Папка содержит все основные подходы к обучению линейных моделей:
* **Теория и База (`01_theory_and_basics`)**: Объяснение того, как работает функция потерь MSE на примере зависимости цены дома от площади.
* **Ручной градиентный спуск (`02_manual_gradient`)**: Алгоритм минимизации ошибки градиентным методом с нуля без использования готовых библиотек машинного обучения.
* **Анализ рекламы (`03_lin_marketing`)**: Модель множественной регрессии для прогнозирования эффективности рекламных расходов.
* **Табличные данные (`04_tabular_insurance`)**: Предсказание медицинских расходов на основе возраста, пола, ИМТ и региона с применением `StandardScaler` и дамми-кодирования.
* **Регуляризация Lasso (`05_lasso_regularization`)**: Пример обнуления весов незначимых шумовых признаков с помощью L1-регуляризации.
* **End-to-End Пайплайн (`06_end_to_end_pipeline`)**: Реалистичный сценарий очистки данных недвижимости (парсинг площади в м2/кв.м, приведение типов, фильтрация аномалий) и вычисление цен на новых данных.
* **Регуляризация и мультиколлинеарность (`07_regularization`)**: Сравнение различных видов регуляризации (L1-Lasso, L2-Ridge, ElasticNet) на зашумленных данных с выраженной мультиколлинеарностью признаков.

### 2. Классификация (`Dima/ml/classification/`)
* **Кредитный скоринг (`bank_credit_scoring`)**: Скрипт подготавливает данные о заемщиках банка, исправляет неверный возраст, заполняет пропуски медианами, обучает логистическую регрессию и выдает решение о выдаче кредита с расчетом метрик F1 и ROC-AUC.
* **Выявление мошенничества (`fraud_detection`)**: Демонстрирует работу с несбалансированными выборками (фрод составляет всего 10%). Используется взвешенная логистическая регрессия (`class_weight='balanced'`) и кривая Precision-Recall для подбора оптимального порога классификации.

### 3. Интерактивные Блокноты (`Dima/marimo_notebooks/`)
Все файлы в этой папке запускаются через Marimo:
```bash
marimo edit Dima/marimo_notebooks/path_to_file.py
```
Они включают в себя пошаговые этапы фильтрации, агрегации данных, работу со слайдерами (`logic.py`), а также построение визуализаций на реальных базах данных (`nba_database.py`).

### 4. Инфраструктура Баз Данных (`Dima/database_sql/`)
* Скрипт `create_db.py` позволяет автоматически восстановить/создать локальную БД SQLite `database.db` с правильными связями внешних ключей (`FOREIGN KEY`) и импортировать информацию о клиентах из файла `Заказчики.json`.
* Файл `notebook.ipynb` содержит упражнения по написанию SQL-запросов для подсчета суммарных покупок с учетом клиентов без заказов (`LEFT JOIN` + `COALESCE`).
